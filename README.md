# Food Ordering System (C++)

A console-based Food Ordering System developed using C++. This project allows users to browse a food menu, add items to a cart, generate bills, and place orders.

## Features

* View Food Menu
* Add Items to Cart
* View Cart
* Generate Bill
* Place Order
* Store Orders in File
* Console-Based Interface

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL (Vector)
* File Handling

## Project Structure

```text
FoodOrderingSystem
│
├── main.cpp
├── orders.txt
└── README.md
```

## How to Run

### Compile

```bash
g++ main.cpp -o food
```

### Run

```bash
./food
```

For Windows:

```bash
food.exe
```
